# Ideas 
Ideas to grab from if I get bored:

- Make it so that the self destruct rooms require a key that's hidden somewhere on a different level of the station.  
- Make it so you have to find a code from a datapad to use the self destruct terminal. 
- Make it so terminals can be hacked to circumvent the need for a code to interact with it. 
- Add a win condition that involves resucing a random number of crew or even specific npcs
- Crew need jobs and schedules.  Go to a station, go to the mess hall, go to bed.   These get interrupted by the scenario.   See crew_ai.md
- Scriptable room generators so we can have rooms that generate with more detail.  
- Demon incursion scenario.  Think Event Horizon.   Must shut down experimental gateway to stop the demons from coming through.  Or maybe you don't.
- The Thing-like scenario.   Monsters look like crew until followed closely.   
- Xenomorph-like/Alien bees/wasps scenario.   Like Aliens but instead it's parasitic wasps that pupate in hosts.   Queen and all that still exists.   
- Spacewalk and airlocks.  It would be cool if we could go outside the space station to work on things like the satelite dishes or to by pass a dangerous corridor.   Bonus if we can launch things into space.
- Life support.  Per tile oxygen count or global oxygen?  Depends on how deep into the sim I want to go.   Should be able to shut off oxygen from a terminal.   Having a space suit lets you ignore the effects of station running out of oxygen.
- Maybe change the temperature too?  Turn off heat to freeze a part of the station?   Turn up the heat to cook an area out?  Maybe turn off the cooling to part of the station that generates heat so that it thermal shut downs or explodes?
- Roguelike style variations.   "Evolved" or "Advanced" versions of enemies.  Broken, improved, or modified versions of weapons and armor.   Useless stuff found in lockers that can be taken to a replicator and turned into something like ammo or parts to improve gear.  
- Burning tiles, exploding barrels, fuel spills, etc.
- Environmental dangers like broken panels/walls that may shock you.
- Security system gone haywire that mistakes you for an intruder.  (Can be fixed/shut down)

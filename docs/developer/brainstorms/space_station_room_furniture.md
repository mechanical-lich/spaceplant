
# Space Station Room Furniture & Props

This file lists common furniture, equipment, and props useful when populating each room type.

## Habitation
- Crew Quarters: bunk beds, personal lockers, fold-out desks, comm panels, porthole/window, ambient lighting, privacy curtains, wall storage, oxygen vents
- Officers' Suites: single bed, wardrobe, private desk, holo-display, small couch, private comms, art/decoration, safe
- Captain's Quarters: queen bed, office desk, navigation console, private comms, personal library, trophy/display case, private bathroom
- Guest Cabins: compact bed, fold table, wall locker, guest comm, small wash basin
- Family Apartments: kitchenette, small table/chairs, sofa, separate sleeping alcoves, storage units, children’s play area
- Childcare / Daycare: padded play mats, toy storage, sleeping cots, safety gates, activity consoles
- Laundry: washers/dryers, sorting bins, folding table, detergent dispensers

## Operations & Command
- Bridge / Command Deck: captain's chair, console banks, tactical displays, star charts, comm arrays, map table, security shutters
- Mission Control / Ops Center: operator consoles, large mission displays, conference table, data-terminals, whiteboards/holos
- Communications Relay: antenna control racks, signal processors, modulation consoles, spare transceivers
- Navigation / Astrometrics: star chart displays, precision scopes, plotting tables, sensor arrays, calibration rigs
- Docking Control: docking monitors, hailing consoles, docking clamps controls, status lights
- Traffic / Flight Ops: flight manifests, launch schedules board, hangar comms, pilot briefing chairs
- Security Office: weapons locker, monitoring wall, detainee log desk, evidence lockers

## Engineering & Systems
- Reactor / Power Core: containment vessel, control console, coolant lines, access catwalks, emergency venting, radiation shielding
- Engineering Workshop: heavy workbench, tool racks, welding rig, vise, spare parts bins, fabrication console
- Maintenance Bay: rolling tool carts, diagnostic panels, parts shelving, maintenance platforms, hoists
- Life Support Control: environmental control panels, scrubber banks, redundancy valves, sensor arrays
- Water / Waste Processing: filtration tanks, pumps, piping manifolds, maintenance ladders
- Cargo Holds: pallet racks, cargo nets, forklifts/rovers, tie-down points, hazard signs
- Utility Tunnels / Service Corridors: conduit trays, pipe runs, junction boxes, access hatches
- Fuel Storage / Propellant Farm: tanking manifold, pressure gauges, safety cutoffs, inerting systems
- Vacuum Airlock (EVA Bay): pressure locks, suit racks, airlock controls, decompression chamber, EVA tool lockers
- Microgravity Work Chamber: tether anchors, restrained workstations, tool holsters, orientation markers

## Science & Research
- General Laboratory: lab benches, sample racks, analysis instruments, fume hood, containment cabinets
- Biolab / Xenobiology Lab: bio-safety cabinets, specimen tanks, containment seals, decontamination showers, incubators
- Chemistry Lab: reagent cabinets, sink with solvent disposal, fume hoods, glassware racks, emergency shower
- Materials / Fabrication Lab: 3D printers, CNC machines, raw material bins, safety shields, inspection microscopes
- Observatory / Sensor Dome: telescope mount, sensor consoles, adjustable optics, vibration isolation mounts
- Medical Research Lab: sterile benches, centrifuges, cold storage, microscopes, specimen freezers
- Environmental / Hydroponics Lab: grow racks, LED arrays, nutrient tanks, irrigation manifolds, climate controls
- Data Center / Server Vault: server racks, cooling ducts, UPS banks, fiber patch panels, access control

## Medical & Safety
- Med Bay / Infirmary: exam beds, medkits, diagnostic console, vitals monitors, privacy curtains
- Surgery / Operating Theater: operating table, surgical lights, instrument trays, sterilization unit, anesthesia console
- Isolation / Quarantine Ward: sealed rooms, negative-pressure fans, PPE stations, observation windows
- Pharmacy: drug cabinets, refrigerated storage, compounding bench, inventory terminal
- Trauma / ICU: life support rigs, ventilators, infusion pumps, secured monitors, immobilization straps
- Morgue / Cryostasis: refrigerated slabs, identification tags, cryo pods, containment seals
- Emergency Shelter / Safe Room: reinforced walls, emergency rations, radio, life support backup, fire suppression

## Logistics & Industry
- Manufacturing Hangar: large assembly platforms, cranes, material feeders, storage bays, welding stations
- Robotics / Drone Bay: docking pads, charging stations, maintenance rigs, spare manipulators
- Freight Airlock: heavy-duty doors, cargo lifts, transfer ramps, clamp fixtures
- Distribution Hub / Freight Sorting: conveyor belts, sorting bins, label printers, inventory terminals
- Storage Lockers / Vaults: secure lockers, vault doors, anti-tamper sensors, shelving
- Customs / Inspection: scanning rigs, inspection tables, quarantine seals, paperwork kiosk

## Commerce & Social
- Galley / Mess Hall: serving line, long tables/benches, food warmers, tray return, waste compactor
- Restaurant / Fine Dining: table seating, hostess station, kitchen pass, ambient lighting, reservation terminal
- Market / Bazaar: vendor stalls, display shelves, credit terminals, awnings
- Bar / Cantina: bar counter, stools, drink fridges, jukebox/entertainment, booths
- Shops / Repair Stalls: display cases, workbench, parts bins, payment terminal
- Bank / Credit Exchange: teller booths, secure safes, verification terminals, queue barriers
- Station Administration / Permits Office: desks, filing cabinets, queue kiosk, holographic forms terminal

## Recreation & Culture
- Gym / Fitness Center: exercise rigs, treadmill/centrifuge, free weights, maintenance mats
- Meditation / Chapel: cushions, altar/console, soft lighting, small shrine, ambient sound emitter
- Library / Archives: stacks/shelves, reading tables, archive lockers, holo-kiosks, climate control
- Theater / VR Dome: seating, projection system, VR rig, stage area, control booth
- Observation Deck / Promenade: benches, telescopes, railing, informational plaques, planters
- Parks / Arboretum: planted beds, irrigation trays, walkways, seating, artificial sunlight panels
- Games Room / Casino: gaming tables, slot machines, prize counter, security cameras

## Security & Law
- Brig / Holding Cells: secured cells, surveillance camera, detainee bench, restraint points
- Interrogation Room: observation mirror/window, table/chairs, recording equipment, stress lighting
- Forensics Lab: evidence lockers, analysis bench, imaging station, contamination gear
- Customs & Immigration: inspection desks, biometric scanners, stamp/print station, holding queue
- Courtroom / Tribunal: judge's bench, witness stand, counsel tables, public seating, record console

## Special / Exotic
- AI Core / Mainframe: server array, central core housing, maintenance platforms, cooling loops, optical interfaces
- Artifact Vault: reinforced containment, inerting locks, environmental controls, anomaly sensors
- Cloning / Genetic Bay: cloning tanks, sample stores, sterile suits, monitoring consoles

- Experimental Drive Lab: test rig, diagnostic consoles, isolation barriers, emergency cutoff
- Observation Platform / Sentry Nest: turret mount, sensor pods, ammo racks, reinforced viewport
- Memorial / Monument Hall: plaques, display cases, floral stands, memorial holograms

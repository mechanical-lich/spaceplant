## Getting Started

### Stations and Player Runs

Space Plants uses a **CDDA-style** save structure. Stations and player runs are separate:

- A **station** is a generated space station — its layout, entities, and state persist independently of any player.
- A **player run** is one character's life on a station. You can have multiple player runs on the same station.

When you launch the game you will see the title screen:

| Option | What it does |
|--------|-------------|
| New Station | Generate a fresh station (you name it or randomise), then create a character |
| Browse Stations | Pick an existing station to start a new player on or continue an existing one |

### Character Creation

After selecting a station you will create your character:

1. **Name** — enter any name, or leave blank for "Unnamed"
2. **Stats** — distribute 10 bonus points across PH, AG, MA, CL, LD, HTCS, CS
3. **Class** — defines stat bonuses, starting equipment, and unlockable skills (see [Classes](classes.md))
4. **Background** — grants one extra starting skill
5. **Confirm** — review and start

### Permadeath

Space Plants is a **permadeath** game. When your character dies:

- Your character's body is converted to a **corpse** and left on the station at the tile where they fell.
- Your player run save is removed from the active player list.
- The **station is saved** intact — all its entities, loot, and your corpse persist.
- You (or anyone else) can start a **new player run** on the same station and potentially find and loot your old character's corpse.

The game auto-saves every 5 turns. You can also save manually from the pause menu (Escape).
